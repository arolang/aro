# Support

Need help with ARO? We're happy to assist!

## Where to Get Help

| Channel | Use For |
|---------|---------|
| [GitHub Issues](https://github.com/KrisSimon/aro/issues) | Bug reports, feature requests |
| [GitHub Discussions](https://github.com/KrisSimon/aro/discussions) | Questions, ideas, show & tell |

**Note**: Only public GitHub Issues and Discussions will be answered. We don't provide private support via email or direct messages.

## What We Can Help With

- The ARO language (syntax, semantics, proposals)
- CLI tools (`aro run`, `aro build`, `aro check`)
- IDE extensions and editor support
- Examples and documentation
- General questions about the project

## Tips for Asking Good Questions

Help us help you! When asking a question:

1. **Include code samples** - Share the relevant `.aro` code
2. **Describe what you expected** - What should have happened?
3. **Describe what actually happened** - Error messages, unexpected output, etc.
4. **Include version info** - Run `aro --version` and share the output
5. **Search first** - Check if someone already asked the same question

## Response Times

This is a community project maintained in spare time. We'll do our best to respond, but please be patient!
