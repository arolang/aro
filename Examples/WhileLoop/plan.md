# Build a while loop demo

Create a single-file ARO application that demonstrates while loops with three patterns:

1. **Basic counter** -- Initialize `<count>` to 1. Loop `while <count> <= 3`, log the count, then recompute `<count>` from `<count> + 1`.

2. **Accumulation** -- Initialize `<i>` to 1 and `<total>` to 0. Loop `while <i> <= 5`, accumulating `<total>` from `<total> + <i>` and incrementing `<i>`. Log the final total (15).

3. **Break statement** -- Initialize `<step>` to 1. Loop `while <step> <= 10`, increment step, then `break.` to exit immediately. Log the step value (2).
