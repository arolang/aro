# Build a file metadata demo

Create a single-file ARO application that demonstrates the `Stat` action for retrieving file and directory metadata.

In the `Application-Start` feature set:

1. Create a file path pointing to this example's own `main.aro` file. Use `Stat the <file-info> for the <file: filepath>` to get the file metadata. Log the result.

2. Create a directory path pointing to this example's directory. Use `Stat the <dir-info> for the <directory: dirpath>` to get the directory metadata. Log the result.

Return OK at the end.
