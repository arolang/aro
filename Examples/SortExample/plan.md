# Build a sorting demo

Create a single-file ARO application that demonstrates the `Sort` action for ordering arrays.

In the `Application-Start` feature set:

1. Sort numbers ascending (default): `Sort the <sorted-numbers> for the <numbers>`.
2. Sort numbers descending: `Sort the <descending-numbers: descending> for the <more-numbers>`.
3. Sort strings alphabetically: `Sort the <sorted-fruits> for the <fruits>`.
4. Use the `Order` verb alias: `Order the <ordered-values> for the <values>`.

Log original and sorted arrays for each example.
