# Build a terminal UI examples index

Create a single-file ARO application that serves as an informational index for the Terminal UI examples collection.

In the `Application-Start` feature set, log the available Terminal UI examples (TerminalSimpleMenu, TerminalTaskManager, TerminalSystemMonitor), list the implemented features (shadow buffer optimization, reactive Watch pattern, template filters, interactive actions, thread-safe terminal service), and provide the run commands for each example.
